first install ImageMagick using the link below
ImageMagick installation file - https://download.imagemagick.org/ImageMagick/download/binaries/ImageMagick-7.1.0-13-Q16-HDRI-x64-dll.exe

install required packages using "pip install -r requirements.txt"

run the file using "python qr.py"

if you change the font, you have to adjust the "font_fix" value in config.json and check which value of font_fix works for you font.
